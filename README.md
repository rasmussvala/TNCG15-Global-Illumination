# Ray Tracer in C++

**Run it on your machine:**
1. Clone repo.
2. Create a build directory in the project.
3. Use CMake to generate the project.
4. Select the RayTracing-TNGC15 folder as the source.
5. Choose the build directory created in step 2.
6. Generate.
